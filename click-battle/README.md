# ⚡ Click Battle Game

A fast-paced browser game where players must click as many times as possible within a limited time. The goal is to achieve the highest click score before time runs out.

This app is built for interactive fun and DevOps deployment practice.

---

## 🎮 Features

- Countdown timer
- Click counter
- Real-time score updates
- Simple and responsive UI
- Fast gameplay

---

## ⚙️ Technology Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- Docker

---

```bash

docker build -t click-battle .

docker run -p 80:80 click-battle

```
